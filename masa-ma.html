15. 3. IfA MASA NIP, and MA = 3x - 4, m/M = 50°, mzS = (5y + 7)°, NI = 21 - 2x, and LP = (4y +15)°. find MAand/P.​


Answer:

Since AMAS is a triangle, we can use the fact that the sum of the angles in a triangle is 180° to find mzA:

mzA = 180° - m/M - mzS

mzA = 180° - 50° - (5y + 7)°

mzA = 123° - 5y

Next, we can use the fact that the opposite angles in a parallelogram are congruent to find mzN:

mzN = mzA

mzN = 123° - 5y

Since MA is a straight line, we have:

mzP = 180° - mzS

mzP = 180° - (5y + 7)°

mzP = 173° - 5y

Now, using the fact that the opposite angles in a parallelogram are congruent, we have:

mzI = mzP

mzI = 173° - 5y

Finally, we can use the fact that the sum of the angles in a triangle is 180° to find mzM:

mzM = 180° - mzI - mzN

mzM = 180° - (173° - 5y) - (123° - 5y)

mzM = 137° + 10y

Since AM is a straight line, we have:

mzP = mzM

mzP = 137° + 10y

Now we can set up a system of equations to solve for x and y:

3x - 4 = 180° - (4y + 15)°

21 - 2x = 50°

mzP = 137° + 10y

Solving the second equation, we get:

21 - 2x = 50°

-2x = 50° - 21

-2x = 29

x = -14.5

Substituting x = -14.5 into the first equation, we get:

3(-14.5) - 4 = 180° - (4y + 15)°

-43.5 - 4 = 180° - 4y - 15°

-47.5 = 165° - 4y

4y = 165° + 47.5

4y = 212.5

y = 53.125

Therefore, MA = 3x - 4 = 3(-14.5) - 4 = -46.5 and P = NI = 21 - 2x = 21 - 2(-14.5) = 50.

17. 3. IfA MASA NIP, and MA = 3x - 4, m/M = 50°, m2S = (5y + 7)°, NI = 21-2x, and LP = (4y +15)°. find MAandzP.​


Answer:

We need to find the values of MA and angle zP. To do this, we can use the following steps:

Use the fact that m/M = 50° to find the value of angle PMA:

m/M = m/P + m/MA

50° = (5y + 7)° + m/MA

m/MA = 50° - (5y + 7)°

m/MA = 43° - 5y

We can also use the fact that angle PMS is a straight line to find the value of angle PMS:

m2S = 180° - m/LP

(5y + 7)° = 180° - (4y + 15)°

y = 22°

Substitute the value of y into the equation for m/MA:

m/MA = 43° - 5y

m/MA = 43° - 5(22°)

m/MA = -77°

Therefore, we have:

m/MA = -77°

Use the fact that NI = 21 - 2x and angle PMA = angle PNI to find the value of x:

m/PNI = m/PMA

(3x - 4)°/(21 - 2x)° = (43° - 5y)°/2

(3x - 4)/(21 - 2x) = (43 - 5y)/2

(3x - 4)/(21 - 2x) = (43 - 5(22))/2

(3x - 4)/(21 - 2x) = -47/2

6x - 8 = -47(21 - 2x)/2

6x - 8 = -987 + 47x

41x = 979

x = 23.87° (rounded to two decimal places)

Substitute the value of x into the equation for MA:

MA = 3x - 4

MA = 3(23.87°) - 4

MA = 61.61° (rounded to two decimal places)

Therefore, we have:

MA = 61.61°

Use the fact that angle PMA + angle MNI + angle zP = 180° to find the value of angle zP:

angle zP = 180° - angle PMA - angle MNI

angle zP = 180° - 43° + 5y - (21 - 2x)

angle zP = 180° - 43° + 5(22°) - (21 - 2(23.87°))

angle zP = 98.46° (rounded to two decimal places)

Therefore, we have:

angle zP = 98.46°

27. Sinasabing ito ang pinakamabili sa masa o karaniwang tao sapagkat mas naiintindihan nila ang wikang ginagamit dito.


Answer:

wika

Explanation:

Ang wika ay isang napakahalagang instrumento ng komunikasyon. Mula sa pinagsama-samang

makabuluhang tunog, simbolo, at tuntunin ay nabubuo ang mga salitang nakapagpapahayag ng

kahulugan o kaisipan. Ito ay behikulong nagagamit sa pakikipagsusap at pagpaparating ng mensahe sa

isa't isa. Nagkakaintindihan tayo, nakapagbibigayan tayo ng ating mga pananaw o ideya, opinyon,

kautusan, tuntunin, impormasyon, gayundin ng mga mensaheng tumatagos sa puso at isipan ng ibang

tao, pasalita man o pasulat gamit ang wika.

Ang salitang Latin na lingua, na may kahulugang "dila'' at "wika" ang pinagmulan ng salitang

Pranses na langue na nangangahulugan ding dila at wika. Kalaunan ito'y naging language na ginawang

batayan din ng parehong salita sa Ingles. Sa maraming wika sa buong mundo, ang mga salitang wika

at dila ay may halos magkaparehong kahulugan. Ito marahil ay sa dahilang ang dila ay konektado sa

pasalitang pagbigkas dahil ang iba't ibang tunog ay nalilikha sa pamamagitan ng iba't ibang posisyon

ng dila. Kaya naman ang wika ay may tradisyonal at popular na pagpapakahulugang sistema ng

arbitraryong vokal-simbol na ginagamit ng mga miyembo ng isang pamayanan sa kanilang

pakikipagtalastasan at pakikipag-ugnayan sa isa't isa.
