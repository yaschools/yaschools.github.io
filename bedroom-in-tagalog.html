Answer first the number of one-bedroom apartments, then the number of two-bedroom apartments that were rented out.


One-Bedroom unit: x
Two-bedroom unit: y

Equation (1):

x + y = 12
y = 12-x

Equation (2):

3,600x + 4,500y = 49,500

Solve the number of one-bedroom units by substitution:

3,600x + 4,500y = 49,500
3,600x + 4,500 (12 - x) = 49,500
3,600x -4,500x + 54,000 = 49,500
-900x = 49,500 - 54,000
900x/900 = 4,500/900
x =  5  ⇒  Number of units of One-bedroom

Solve for y, the two-bedroom unit:
y = 12-x
y = 12 - 5
y = 7  ⇒   Number of units of Two-bedroom

Check:

Number of Units:
x + y = 12
5 + 7 = 12
12 = 12 (True)

Rent amount:
3,600x + 4,500y = 49,500
3,600 (5) + 4,500 (7) = 49,500
18,000 + 31,500 = 49,500
49,500 = 49,500   (True)

22. you plan to create a floor plan of your dream house.l two of the rooms are the masters' bedroom and an extra bedroom. if the are of the masters' bedroom is x^2-255 sq. inches and the are of the extra bedroom is x^3-216 sq. inches, 1. what are the dimensions of the : (show the steps, label properly) a. masters bedroom b. extra bedroom 2. if you will give a value of x to exactly compute the dimensions, what is it and why?


Problems Involving Algebraic Equations

The dimensions of the masters bedroom is 15 inches by 15 inches and the extra bedroom is 6 inches by 6 inches. The value of x is 15 for x² - 225 and 6 for x³ - 216.In solving algebraic equations factoring can be used. For the given problem, factoring using difference of two squares is applied to x² - 225. For  x³ - 216, it is applicable to use factoring using difference of two cubes.

Solutions:

a. Given: x² - 225            

Factor x² - 225.

x² = (x)(x)

-225 = (15)(-15)

Using the factors above, find the dimensions of the masters bedroom.

x² - 225 = (x + 15)(x - 15)

Let: x + 15 = 0.

x + 15 - 15 = 0 - 15

x = -15

Let: x - 15 = 0

x - 15 + 15 = 0 + 15

x = 15

Since the dimensions of any given figure is positive, we will take 15. Thus, the dimensions of the masters bedroom is 15 inches x 15 inches.

b. Given:  x³ - 216

Factor x³ - 216.

x³ = (x)(x)(x)

-216 = (6)(-6)(6)

Using the factors above, find the dimensions of the extra bedroom.

x³ - 216 = (x - 6)(x² + 6x + 36)

Let: x - 6 = 0

x - 6 + 6 = 0 + 6

x = 6

Let: x² + 6x + 36 = 0

Since it is not a perfect square trinomial, we will use the quadratic formula to solve for x.

a = 1, b = 6, c = 36

x = [tex]\frac{-b + \sqrt{b^2 - 4ac} }{2a}[/tex]

x = [tex]\frac{-6 + \sqrt{(6)^2 - 4(1)(36)} }{2(1)}[/tex]

x = [tex]\frac{-6 + \sqrt{36 - 144} }{2}[/tex]

x = [tex]\frac{-6 + \sqrt{-108} }{2}[/tex]

x = [tex]\frac{-6 + \sqrt{-108} }{2}[/tex] or x = [tex]\frac{-6 - \sqrt{-108} }{2}[/tex]

Since the dimensions of any given figure is positive, we will take 6. Thus, the dimensions of the extra bedroom is 6 inches x 6 inches.

27. A housing project has 1822 units. There are twice as many two-bedroom units as there are one-bedroom units, and 250 fewer three-bedroom units than one-bedroom units. How many of each kind of units are there?with full solution pls ​


Let's use some variables to represent the number of each type of unit:

Let x be the number of one-bedroom units.

Then, the number of two-bedroom units would be 2x (twice as many as one-bedroom units).

And, the number of three-bedroom units would be x - 250 (250 fewer than one-bedroom units).

The total number of units is 1822, so we can write an equation:

x + 2x + (x - 250) = 1822

Simplifying and solving for x:

4x - 250 = 1822

4x = 2072

x = 518

So there are 518 one-bedroom units, 2x = 1036 two-bedroom units, and x - 250 = 268 three-bedroom units.

Therefore, there are 518 one-bedroom units, 1036 two-bedroom units, and 268 three-bedroom units.
