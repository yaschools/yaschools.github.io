28. solb the following problems pa answer nmn po sana tama po tenchuu:))))​


=============================

What is asked?

How much will it cost if I give 500 in the total of my purchase?

What are the given facts?

4 items i bought worth: Php 39.90, Php 68.60, Php 58.75, Php 129.25[tex].[/tex]

Solve the hidden given:

Php 39.90 + Php 68.60 + Php 58.75 + Php 129.25 = N - Php 500 = N

PLAN:

What operations will be used?

Addition and subtraction

Solve:

Php 39.90 + Php 68.60 = 108.5

Php 58.75 + Php 129.25 = 188

108.5 + 188 = 296.5

Php 500 - Php 296.5 = 203.5

Check:

108.5 + 100 = 296 -500 = Php 203.5

Therefore, the exchange rate is 203.5.
