Saranggola

Habang ang pagpapalipad ng saranggola ay masaya, ang pagpapalipad nito ay hindi kasingdali ng tila. Kapag ang isang bata ay nabigong magpalipad ng saranggola, ito ay maaaring magalit sa kanya.

Ang mga kondisyon ng hangin at hugis ng saranggola ay may malaking papel, ngunit ang pinakamahalagang kadahilanan ay ang wastong pamamaraan.

Piliin ang perpektong hugis ng saranggola

Ang mga karaniwang saranggola ay mahusay para sa mga nagsisimula. Bagama't ang mga bata ay madalas na naghahanap ng mas mapaghamong bagay at maaaring pumili ng mas malaki o mas mahilig sa saranggola. Ngunit tandaan, ang bawat saranggola ay nag-aalok ng ibang karanasan sa paglipad.

Kung mahina ang hangin, humigit-kumulang 6 hanggang 15 milya bawat oras, kung gayon ang mga saranggola ng brilyante, delta at dragon ay magiging perpekto. Kung malakas ang hangin, tulad ng 15 hanggang 25, kung gayon ang isang kahon o Parafoil kite ang pinakamagandang saranggola para sa mga bata.

Suriin ang lagay ng panahon bago magpalipad ng saranggola

Ang mga kondisyon ng panahon ay maaaring gumawa o masira ang mga plano ng pagpapalipad ng saranggola ng isang bata.

Kung mahangin ang panahon ngunit hindi masyadong malakas, ito na ang tamang oras para magpalipad ng saranggola. Sa mahinang simoy ng hangin, ang mga bata ay maaaring magpalipad at sumayaw ng mga saranggola.

Tukuyin ang tamang lugar

Pagdating sa pagpapalipad ng saranggola, mahalagang hanapin ang isang lugar na may mas maraming open space na katumbas. Halimbawa, ang mga parke, beach, field at field ay ang pinakamahusay na pagpipilian. Ang mga puno ay hindi mapanganib, ngunit maaari silang 'kumain' ng saranggola.

Gumawa ng ilang distansya sa saranggola at paliparin ito

Mahalagang bantayan ang hangin bago lumipad, at hilingin sa glider na pakawalan ang saranggola. Kapag nasa itaas ang saranggola, maaari itong laruin ng bata. Kailangan din niyang hilahin ang lubid para magbigay ng kaunting alitan.
