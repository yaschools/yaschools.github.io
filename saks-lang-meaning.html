2. Hola como estas sak pase say n'ap boule meaning??


"Sak Pase" is a common Haitian Creole phrase that means "What's happening? It is often used to greet friends, similar to how you would say "what's up" in English. The expected response is "N'ap boule." Which literally translates "we're burning" but it actually means "we're hanging out."

D= (3)² - 4(5)(-6) = 9 + 120 = 129

3. 6x² - 4x + 10 = 0

D= (-4)² - 4(6)(10) = 16 - 240 = -234

4. 5x² + 3x + 8 = 0

D= (3)² - 4(5)(8) = 9 - 160 = -151

5. 3x² + 2x + 10 = 0

D= (2)² - 4(3)(10) = 4 - 120 = -116

6. 4x² + 5x + 1 = 0

D= (5)² - 4(4)(1) = 25 - 16 = 9

7. 5x² - 4x + 7 = 0

D= (-4)² - 4(5)(7) = 16 - 140 = - 124

8. 6x² + 7x + 12 = 0

D= (7)² - 4(6)(12) = 49 - 288 = -239

9. 5x² + 4x - 9 = 0

D= (4)² - 4(5)(-9) = 16 + 180 = 196

10. x² + 4x + 5 = 0

D= (4)² - 4(1)(5) = 16 - 20 = -4
