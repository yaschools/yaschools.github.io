6. bakit kailangang matutuhan ang elemento at prinsipyo ng sining biswal?


Answer:

bilang isang manlilikha ng sining, mahalagang matutunan ang mga elemento ng sining biswal sapagkat ito ang bumubuo sa isang obra. ang prinsipyo ng sining naman ay tumutukoy sa kung papaano ginagamit ng isang manlilikha ang mga elemento ng sining upang makalikha ng obra.

Explanation:

ang mga elemento ng sining ay: line, shape, value, form, color, texture, at space

ang mga prinsipyo ng sining ay: balance, contrast, proportion, emphasis, movement, pattern, rhythm, at harmony
