22. ano ang kahulugan ng duruan ano ang kahulugan ng mainam ano ang kahulugan ng namalas ano ang kahulugan ng tumambad ano ang kahulugan ng himpapawid ano ang kahulugan ng dakip ano ang kahulugan ng hapo ano ang kahulugan ng nasa


Ang mga kahulugan ng mga sumusunod na mga termino:

Duruan- isang bagay na pantusok sa iihawing karne ng baboy, manok o isda at iba pa. Isa itong bagay na may matulis ang isang dulo nito.

Mainam- nakasasapat o nakaaabot sa isang pamantayan o inaasahan sa kaniya. Maaaring tumukoy sa kalidad  o sukat ng isang bagay o personalidad.

Namalas- tumutukoy sa pananaw o pagtingin ng isa sa isang isyu o bagay. Ito ay nasa pangnagdaang kapanahunan ng terminong pangmalas.

Tumambad- nangngahulugang nasa mismong harap mo ang isang pangyayari, nakita o natunghayan mo ito ng aktuwal. Laging ang reaksyon nito ay hindi inaasahan at dramatiko.

Himpapawid- ibang tawag sa kalangitan o ang mga nakikita sa itaas dito sa ating atmospera o sa labas ng Lupa.

Dakip- isinasalin din bilang huli o aresto. Tumutukoy ito sa mga tao, bagay o hayop na sinikap o puwersahang kinuha mula sa pagtakas.

Hapo- ginagamit kapag ang isa ay pagod na o mabilis mapagod.

Nasa- ay ang salitang-ugat para sa pagnanasa. Maaari ding gamiting ibang salita dito ang nais, gusto, motibo ng isa. Ang mga nasa ng puso ay hindi naman laging masama. Pero madalas na ginagamit ito sa maling mga nasa o pagnanasa. Nakadepende na lamang sa kabuoang kahulugan ng pagkakagamit.


23. ano ang kahulugan ng blockano ang kahulugan ng microphone ano ang kahulugan ng viewer ano ang kahulugan ng post o upload ano ang kahulugan ng mydayano ang kahulugan ng share ano ang kahulugan ng block o unfriendano ang kahulugan ng browseano ang kahulugan ng video callano ang kahulugan ng online o offline ano ang kahulugan ng loadano ang kahulugan ng downloadano ang kahulugan ng cellphone ano ang kahulugan ng apps​


I-block: upang pigilan ang isang tao o isang bagay na ma-access o makipag-ugnayan sa isang tao o system.

Mikropono: isang aparato na nagko-convert ng mga sound wave sa isang electrical signal para sa pag-record o amplification.

Manonood: isang taong nanonood o tumitingin sa isang bagay, gaya ng video, larawan, o dokumento.

Mag-post o Mag-upload: upang mag-publish o magdagdag ng nilalaman, tulad ng teksto, mga larawan, o mga video, sa isang website o online na platform.

MyDay: isang feature sa ilang social media platform na nagpapahintulot sa mga user na magbahagi ng mga update at post tungkol sa kanilang araw o aktibidad.

Ibahagi: upang bigyan o payagan ang ibang tao na magkaroon ng isang bagay, tulad ng impormasyon, media, o mga mapagkukunan.

I-block o I-unfriend: upang alisin o paghigpitan ang pag-access o pakikipag-ugnayan ng isang tao sa isang tao o system, kadalasan dahil sa mga negatibong pakikipag-ugnayan o mga alalahanin sa privacy.

Mag-browse: upang maghanap, tingnan, o tuklasin ang online na nilalaman, gaya ng mga website, larawan, o produkto.

Video call: isang real-time na komunikasyong video sa pagitan ng dalawa o higit pang tao, kadalasang gumagamit ng computer o mobile device.

Online o Offline: tumutukoy sa estado ng pagiging konektado o hindi nakakonekta sa internet o isang network.

Mag-load: upang maglipat o magdala ng data o nilalaman mula sa isang server o storage device patungo sa isang computer o device.

I-download: upang ilipat o i-save ang data o nilalaman mula sa isang server o storage device patungo sa isang computer o device.

Cellphone: isang mobile device na maaaring tumawag, magpadala ng mga text, at mag-access sa internet at iba pang mga application.

Mga App: maikli para sa "mga application", mga program o software na idinisenyo upang magsagawa ng mga partikular na function o gawain sa isang computer o mobile device.
