19. Tatlong uri ng kasinungalingan


Tatlong Uri ng Kasinungalingan:

Ang tatlong uri ng kasinungalingan ay jocose lies, officious lies, at pernicious lies. Ang jocose lies ay kasinungalingan upang maghatid ng kasiyahan. Ang officious lies ay kasinungalingan upang makalikha ng usaping mapagbabalingan. Ang pernicious lies ay kasinungalingan upang makasira ng reputasyon ng isang tao upang pumabor sa iba.

Ang jocose lies ay kasinungalingang hindi sinasadya. Ito ay ginagawa upang maghatid ng saya tulad ng pagsasabing si Santa Claus ay nagbibigay ng aginaldo sa mga batang naging mabuti sa buong taon.

Ang officious lies ay kasinungalingang sinadya upang mapagtakpan ang kamaliang ginawa tulad ng pagtanggi sa pagkuha ng salapi gayong ang salapi ay ginamit na pansugal.

Ang pernicious lies ay kasinungalingang lubhang mapaminsala sapagkat ito ay maaaring makasira ng reputasyon ng isang tao tulad ng pagkakalat ng mali at malisosyang balita ukol sa kaibigan.

Keywords: kasinungalingan, jocose lies, officious lies, pernicious lies

29. 3 uri ng kasinungalingan


Answer:

1. Lies of Commission - ito ay isang uri ng kasinungalingan na kung saan ang mga tao ay nagsasabi ng mga bagay na hindi naman totoo.

2. Lies of Omission - ito ay isang uri ng kasinungalingan na kung saan ang mga tao ay hindi nagsasabi ng kumpletong detalye.

3. Character lies o lies of influence - ito ang pagsasabi ng mga bagay na walang direktang kinalaman sa mga katotohanan para pagtakpan ang mga bagay.
