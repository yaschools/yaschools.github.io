23. she has received a package a while ago


IS THE SENTENCE CORRECT? YES/NO.ANSWER: YES

The sentence "She has received a package a while ago." is a correct sentence. Why? The sentence is clearly in the Present Perfect form. In the sentence, the helping verb has was used alongside with the main verb/past participle received.

WHAT IS PRESENT PERFECT TENSE?  

It is a tense of verb used to show that an action is done or taken place once or many times before the current time. It uses the helping verb "has" alongside with the main verb if the subject is in singular form and uses "have" alongside with the main verb if the subject is in plural form.

Analyze the sentence one by one and determine each one's part of speech.

She - Pronoun, used as the main subject.has - Modal Verb, used as helping verb. Makes the sentence in Present Perfect form.received - Main Verb/Past Participle, used as the main verb that describes the action of the subject "She".a package - Direct object, used as the receiver of the verb "received".a while ago - Adverb of Time, used to show the time when the package is received. (Also shows that the subject performed the verb only once)

These shows that the sentence given was in correct form.

HOWEVER,

The sentence can be transformed into much simpler form. You can transform the sentence from Present Perfect Tense form to Past Tense form. All you have to do is to omit the helping verb has.

COMPARISON

She has received a package a while ago. - Present Perfect TenseShe received a package a while ago. - Past Tense

For Better Understanding, you can translate the sentence into Tagalog to see the difference between the two.

She has received a package a while ago. - Siya ay nakatanggap ng pakete kanina lang.

She received a package a while ago. - Nakatanggap siya ng pakete kanina lang.
