13. 14.) What would be the name of the congruent triangles given their corresponding sides: LO RP, MO QP, LM RQ? A. ALOM B. 40ML AQPR APQR C. AMOL ARQP D. ALMO APQR​


Answer:

The name of the congruent triangles would be option D, ALMO APQR.

The name of a triangle is typically written with the vertices listed in a specific order. One common convention is to list the vertices in alphabetical order. In this case, the corresponding sides of the triangles are LO and RP, MO and QP, and LM and RQ.

Starting with the first vertex in alphabetical order, we can see that L corresponds to both triangles. The next vertex is M, which corresponds to both triangles as well. The third vertex in alphabetical order is O, which corresponds to the first triangle, and P, which corresponds to the second triangle. The final vertex is R, which corresponds to both triangles.

Putting these vertices in alphabetical order, we get the name ALMO for the first triangle and APQR for the second triangle. Since the triangles are congruent, we can write them as ALMO ≅ APQR or APQR ≅ ALMO.
