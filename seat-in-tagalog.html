3. A concert is held in a hall with a seating capacity of 800 seats. There are three types of seats: S seats, A seats, and B seats. There are 200 more A seats than S seats and 4 times as many B seats as S seats. The price per seat is 80 dollars for S seats, 70 dollars for A seats, and 50 dollars for B seats. If all seats are sold out, the total sales will be ______ dollars.


To properly analyze the problem,

Let x = S seats with 80 dollars per seat
y = A seats with 70 dollars per seat
z = B seats with 50 dollars per seat
T = seating capacity = 800 seats

Condition 1: There are 200 more A seats than S seats

y = x + 200

Condition 2: 4 times as many B seats as S seats

z = 4x

Solving for the number of S seats through substitution,

800 = x + (x+200) + 4x
800-200 = 6x
600 = 6x
x = 100 seats = S seats

Finding for the number of A seats, given Condition 1:

y = 100 + 200
y = 300 seats = A seats

Finding for the number of B seats, given Condition 2:

z = 4(100)
y = 400 seats = B seats

Now that you have all the number of seats for S, A, and B, respectively. You can now solve for the total number of seats sold:

Seats sold in Dollars = 100(80) + 300(70) + 400(50)
Seats sold in Dollars = $49,000

Therefore, the 800 seats were sold for $49,000.

5. What is the seating capacity of an auditorium with 30 rows of seats if there are 15 seats in the first row, 19 seats in the second row, 23 seats in the third row, and so on?


n = 30 rows
a_1 = 15
d = a_2 - a_1 = 19 - 15 = 4

S_n = (n/2)[2a_1 + (n - 1)d]
S_30 = (30/2)[2(15) + (30 - 1)(4)]
S_30 = 15[30 + 29(4)]
S_30 = 15(30 + 116)
S_30 = 15(146)
S_30 = 2190

There are 2190 seating capacity in the auditorium.

Use Sum of the sequence formula:

Sn = n/2 [ 2(a₁) + (n-1)(d)]

n = 40

a₁ = 25

d = 3    ⇒  28-25 = 3

Substitute these values:

S₄₀ = 40/2 [2(25) + (40-1)(3)]

S₄₀ = 20 [50 + (39)(3)]

S₄₀ = 20 [50 + 117}

S₄₀ = 20 (167)

S₄₀ = 3,340

ANSWER: The seating capacity is 3,340.
