Answer:

Ang mga "Antecedent Strategies" ay ginagamit upang pigilan o pakilosin ang isang partikular na pag-uugali bago ito mangyari. Samantalang ang "Consequent Strategies" ay ginagamit kapag nangyari na ang isang pag-uugali upang maiwasan ang pagpapatuloy o pag-uulit ng pag-uugali, o upang palakasin ang isang inaasahang pag-uugali.

Halimbawa ng Antecedent Strategies ay ang pagpapakita ng positibong halimbawa o pagsasabi ng tamang paraan ng pag-uugali bago pa mangyari ang isang sitwasyon. Halimbawa, bago pumasok sa classroom, ang guro ay maaring magpakita ng tamang pag-uugali tulad ng pag-upo ng maayos at pagpapakita ng respeto sa kapwa estudyante.

Sa kabilang banda, ang mga Consequent Strategies ay ginagamit upang mapigilan o mapanatili ang isang pag-uugali. Halimbawa, kung ang isang estudyante ay nagsasalita nang malakas sa klase, ang guro ay maaaring sabihin sa estudyante na maging tahimik upang hindi maistorbo ang klase. Kung ang estudyante ay tahimik na nang magpatuloy ang klase, ang guro ay maaaring magbigay ng positibong pabuya tulad ng pagpapuri sa estudyante para sa magandang pag-uugali na ito.
