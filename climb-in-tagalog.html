21. A snail is crawling straight up a wall. The first hour it climbs 16 inches, the second hour it climbs 12 inches and the each succeeding hour, it climbs only three-fourths the distance it climbed the previous houra. How far does the snail climb during the seventh hour? With solution


Answer:

55 97/256 inches

Step-by-step explanation:

1st hr = 16"

2nd hr = ( 16 ) ( 3/ 4 ) = 12"

3rd hr = ( 12 ) ( 3 / 4 ) = 9"

4th hr = ( 9 ) ( 3 / 4 ) = 6 3/4"

5th hr = ( 6 3/4 ) ( 3 / 4 ) = ( 27 / 4 ) ( 3 / 4 ) = 81 / 16" or 5 1/16"

6th hr = ( 81 / 16 ) ( 3 / 4 ) = 243 / 64" or 3 51/64"

7th hr = ( 243 / 64 ) ( 3 / 4 ) = 729 / 256" or 2 217/256

16 + 12 + 9 + 6 3/4 + 5 1/16 + 3 51/64 + 2 217/256 = 55 97/256
