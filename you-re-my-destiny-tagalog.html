x+2 = 0, means x = -2

x-1 = 0, means x = 1

x-3 = 0, means x = 3

This means that for the following values of x: -4, -2, 1, and 3, the value of y = 0. In the relation to 0 column, of course, we also put y = 0. Remember that the x-axis is just the line y=0, so for these values of x, the point is on the x-axis.
