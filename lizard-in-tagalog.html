24. a lizard accelerates from 2m/s to 10m/s in 4sec what is the l lizard average acceleration ​


Answer:

The average acceleration of the lizard can be calculated using the formula:

average acceleration = (final velocity - initial velocity) / time

Given that the initial velocity of the lizard is 2 m/s, the final velocity is 10 m/s, and the time taken is 4 seconds, we can plug in the values and calculate the average acceleration:

average acceleration = (10 m/s - 2 m/s) / 4 s

average acceleration = 8 m/s / 4 s

average acceleration = 2 m/s^2

Therefore, the average acceleration of the lizard is 2 m/s^2.

28. At the zoo,the ratio of snakes to lizard is 3:2..if there were 10 lizards,how many snakes would there be?


Step-by-step explanation:

The ratio of snakes to lizards is 3:2, meaning for every 3 snakes, there are 2 lizards. If we know that there are 10 lizards, we can use this ratio to find out how many snakes there are.

To do this, we can set up the equation:

3x = 2(10)

Where x is the number of snakes. We can then solve for x by dividing both sides of the equation by 3:

x = (2(10))/3

x = (20/3)

So if there are 10 lizards, there would be 20/3=6.666... or 6 snakes in the zoo, in this case we can round up the number to 7 snakes.
