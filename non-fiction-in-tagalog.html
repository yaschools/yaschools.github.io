29. the ratio of the number of fiction books to the number of non-fiction books in a library is 5:2. If there are 1421 fiction and non-fiction books altogethee, how many more fiction than non fiction books are there in the library?​


Answer:

There are 609 more fiction books than non fiction books in the library.

Step-by-step explanation:

Ratio Word Problem

What is ratio?

Ratio indicates how many times one number contains another. It shows the the relative sizes of two or more values.

The problem above is a ratio word problem since it is asking for how many more fiction than non fiction books there are in the library based on the ratio of the books.

Given:

ratio of fiction books to non-fiction books - 5:2

total number of books - 1421

Equation:

Let x be the number.

5x + 2x = 1421

Solution:

5x + 2x = 1421

7x = 1421

7x/7 = 1421/7

x = 203

Fiction Books

5x = 5 (203) = 1015

There are 1015 fiction books.

Non-Fiction Books

2x = 2 (203) = 406

There are 406 non-fiction books.

To know how many more fiction than non fiction books there are in the library, we need to subtract.

1015 - 406 = 609

Final Answer:

There are 609 more fiction books than non fiction books in the library.

Checking:

5x + 2x = 1421

1015 + 406 = 1421

1421 = 1421 ✔

Cross multiply.

5:2 = 1015:406

2 × 1015 = 5 × 406

2030 = 2030 ✔
