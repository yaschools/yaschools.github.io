18. Simple interest at 15% per annum for 3 years is PhP 45,000​


The formula for simple interest is:

I = P * r * t

where:

I is the interest amount

P is the principal amount

r is the annual interest rate as a decimal

t is the number of years the amount is invested or borrowed for

Using the given values:

I = 45,000

t = 3

r = 15% = 0.15

We can solve for P:

P = I / (r * t)

P = 45,000 / (0.15 * 3)

P = 45,000 / 0.45

P = 100,000

So, the principal amount is 100,000.

Answer:

The amount earned by Ms. Natalie for the first year is, a = 2,00,000. The increment per annum is, d = 25,000. We have to calculate her earnings in the first 5 years. Hence n = 5. Substituting these values in the sum sum of arithmetic sequence formula,

Sn= n/2 [2a1 + (n-1) d]

⇒ Sn = 5/2(2(200000) + (5 - 1)(25000))

= 5/2 (400000 +100000)

= 5/2 (500000)

= 1250000

She earns $1,250,000 in 5 years. We can use this formula to be more helpful for larger values of 'n'.

27. php 500 00.00 for 5 years at 11.3% per annum


Answer:

The formula for calculating the future value of a sum of money (FV) invested at an interest rate (i) for a number of years (t) is given by the following equation:

FV = PV * (1 + i)^t

Where PV is the present value, or the initial amount invested. In this case, the present value is PHP 500,000.00.

Using the given interest rate of 11.3% per annum, and the number of years of 5, the calculation would be as follows:

FV = 500,000 * (1 + 0.113)^5

FV = PHP 901,134.57

So after 5 years, at an interest rate of 11.3% per annum, the future value of PHP 500,000.00 would be PHP 901,134.57.
