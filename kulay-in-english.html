Gawain 1 - A

asul + dilaw = berde

pula + dilaw = kahel

asul + pula = lila

Gawain 1 - B

pula ( red ) + kahel (orange) = red orange

dilaw (yellow) + berde (green) = yellow green

asul (blue) + lila (violet) = indigo

asul (blue) + berde (green) = blue green
