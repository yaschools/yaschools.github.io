3. IBIGAY ANG KAHULUGAN NG SUMUSUNOD: GENETIC ENGINEERING,ABORSYON, EUTHANASIA,ORGAN DONATION/TRANSPLANTTAGALOG PO SANA ANG SAGOT NOT ENGLISH ​


Answer:

Ang inhenyeriyang henetiko o genetic engineering na tinatawag ring genetic modification ang direktang manipulasyon ng genome ng isang organismo gamit ang biyoteknolohiya.

Aborsiyon ang pagpapalaglag, pagpapaagas o aborsyon[1] ay ang sinadyang pagtatanggal ng embryo o fetus sa loob ng matres ng babae, na nagsasanhi ng kamatayan nito.

Ang eutanasya (Ingles: euthanasia) ay ang pagpapatiwakal ng isang indibidwal na nagnanais ng wakasan ang sariling buhay sa tulong ng ibang tao. itong direktang pagtanggal ng kanilang karapatang mabuhay sa mundo.

Ang transplantasyon ng organo o siruhiya ng pagtransplanta (Ingles: organ transplantation, transplant surgery), literal na "pag-ani, paglipat at muling pagtatanim ng organo") ay ang paglilipat ng isang organo magmula sa isang katawan papunta sa isa pang katawan, o mula sa isang pook ng donasyon (inabuloy na organo) papunta sa isa pang lugar o lokasyon sa sariling katawan ng pasyente, para sa layunin ng pagpapalit ng nawawala o napinsalang organo ng tatanggap.

24. Andy, Ben, and Chris donated P550 altogether. Ben donated P100 more than Andy. Chris donated 3 times than Andy. Find how much did each of them donated.please help!​


Answer:

Andy donated P90

Ben donated P190

Chris donated P270

Step-by-step explanation:

a + b + c = 550

a = ?

b =  a + 100

c = 3a

solution:

a + a + 100 + 3a = 550

simplify:

5a + 100 = 550

subtract 100 from both sides:

5a + 100 - 100 = 550 - 100

5a = 450

Divide both sides by 5:

[tex]\frac{5a}{5}[/tex] = [tex]\frac{450}{5}[/tex]

a = 90

b = (90) + 100 = 190

c = 3(90) = 270
