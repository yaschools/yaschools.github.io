6. If BAD = 10, DAC = 11, and CGI = 22 what is the value of OCCAM?​


Answer:

Since BAD = 10, we know that B+A+D = 10. Similarly, we can find that D+A+C = 11 and C+G+I = 22.

We can use this information to form equations for each of the letters in OCCAM. Let's call the value of O x, since we don't know what it is yet. Then we have:

O+A+C+C+M = ?

We don't know what the sum is yet, but we can substitute the values we do know:

x + D = ?

D + A + C = 11

C + G + I = 22

Now we can use the first two equations to find the value of x:

x + D = ?

B + A + D = 10

We can substitute the second equation to get:

x + (10 - B) = ?

We don't know what B is, but we can see that it cancels out when we subtract the equations:

x + D - (x + 10 - B) = D + A + C

B - 10 + C + M = D + A + C

B + M = D + A + 10

Now we can substitute the third equation:

B + M = C + G + I - D + 10

B + M = 22 - D + 10

B + M = 32 - D

We still don't know what B or D are, but we can see that the sum of B and M must be 32 minus D. The only digits that add up to a number between 20 and 30 are 9 and 1, so either B or M must be 9 and the other must be 1.

Let's assume that B is 9 and M is 1. Then we have:

O + A + C + C + 1 = ?

x + D = ?

D + A + C = 11

C + G + I = 22

B + M = 10

From the last equation, we know that D must be 1. Then we can substitute that value into the other equations to find:

x + 1 = 11

x = 10

C + G + I = 22

B + M = 10

So we have found that OCCAM = 10111.
