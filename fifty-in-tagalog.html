How will you write 0.56 in words?

• Answer:

A. Zero point fifty-six

B. fifty-six tenths

[tex]\boxed{\tt\:C. fifty-six \:hundredths}[/tex]

D. Zero and fifty-six hundredths

• Explanation:

Writing decimal numbers in words, first is to write the verbal phrase for whole numbers. But in the given, since zero is not a whole number, you're not in need to write the verbal phrase for it.

Then the numbers after the decimal point should be named just like that in whole numbers. For example, 56 is fifty-six.

But, at the end you need to write the place value of the last decimal number. The place value of 5 is tenths while 6 is hundredths.
