3. There are 725 carps and some goldfish in an aquarium. There are five times as many carps as goldfish. How many fishes are there in the aquarium?


Answer:

Let's call the number of goldfish "x."

According to the problem, there are five times as many carps as goldfish. So the number of carps is 5x.

We also know that the total number of fish is 725.

Putting this all together, we can write the equation:

x + 5x = 725

Simplifying, we get:

6x = 725

Dividing both sides by 6, we get:

x = 120.83

Since we can't have a fraction of a fish, we need to round this down to the nearest whole number. So there are 120 goldfish.

To find the number of carps, we can use the equation we came up with earlier:

carps = 5x

carps = 5(120)

carps = 600

So there are a total of 120 + 600 = 720 fish in the aquarium.
