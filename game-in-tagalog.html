28. ano ang ibig sabihin ng invasion game?tagalog words


Answer:

﻿Ang invasion game ay binubuo ng sampung manlalaro sa bawat dalawang grupo. Kailangan maraming matay ang mga manlalaro sa bawat panig upang sila ay manalo at masalakay ang base ng kanilang kalaban.

Kadalasan na tinatawag dito ay agawan base o “tag” dahil tinataya ang mga kalahok dito at inuubos ang bawat miyembri ng mga grupo.

Bukod pa rito, nasa gilid ng isang basketball court dapat ang base ng mga grupo upang pantay at patas ang kanilang lugar.

Dito maipapakita ang bilis ng pagtakb ng mga manlalaro upang hindi mataya. Kahit sino ay pwedeng sumali rito at walang limitasyon sa mga edad.
