1. The typical computer screen has a WIMP setup (What does WIMP stand for?)​


Answer:

Windows, Icons, Menus and Pointers.

Explanation:

WIMP is an acronym for Windows, Icons, Menus and Pointers. Alternatively, it is an acronym for Windows, Icons, Mice and Pull-down menus. Examples of user interfaces that are based on the WIMP interaction style include: Microsoft Windows for PCs, MacOs for Apple Macintosh, various X Windows-based systems for UNIX, etc.
