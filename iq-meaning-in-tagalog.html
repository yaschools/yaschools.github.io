To answer this question, we need to use the properties of the normal distribution. We know that the mean is 100 and the standard deviation is 15.

We want to find the percentage of the population with IQ scores greater than 130. We can use a standard normal distribution table or a calculator to find the corresponding z-score, which is:

z = (130 - 100) / 15 = 2

This means that an IQ score of 130 corresponds to a z-score of 2. We can then find the percentage of the population with a z-score greater than 2 using a standard normal distribution table or a calculator.

Looking up the value in the standard normal distribution table, we find that the percentage of the population with a z-score greater than 2 is approximately 2.28%. Therefore, we can conclude that approximately 2.28% of the general population have IQ scores larger than 130.

z = (x - μ) / σ

where:

x = the IQ score

μ = the mean IQ score (100)

σ = the standard deviation (16)

For x = 90:

z = (90 - 100) / 16 = -0.625

For x = 110:

z = (110 - 100) / 16 = 0.625

Using a standard normal distribution table or calculator, we can find the proportion of people with IQs between -0.625 and 0.625:

P(-0.625 < z < 0.625) = 0.4129 - 0.2334 = 0.1795

Therefore, about 17.95% of the general population have IQs between 90 to 110.

The z-score that corresponds to the top 2% is approximately 2.05. Therefore, we can solve for x in the following equation:

2.05 = (x - 100) / 16

x - 100 = 2.05 * 16

x = 132.8

So, the lowest IQ score in the SBIS that would qualify for MENSA membership is approximately 133.
