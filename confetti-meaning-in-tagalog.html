8. For every touchdown scored by the Timberwolves, the mascot does 3 back flips and the cheerleaders set off 6 confetti cannons. How many touchdowns did the Timberolves score if the cheerleaders set off 18 confetti cannons?


Answer:

Timberwolves did 3 touchdowns if the cheerleaders set off 18 confetti cannons.

Step-by-step explanation:

To know the number of touchdowns Timberwolves did if the cheerleaders set off 18 confetti cannons, we need to express the given data in ratios to show the relationship of every touchdown to every set off of confetti cannons.

Let us analyze the given problem.

Given:

"Every touchdown scored by the Timberwolves, the mascot does 3 back flips and the cheerleaders set off 6 confetti cannons."

1:3:6 but because the question is just about the touchdown and confetti cannons, we are just going to show the relationship of every touchdown to every set off of confetti cannons.

= 1:6 or 1/6

"How many touchdowns did the Timberolves score if the cheerleaders set off 18 confetti cannons?"

Let x represents the number of touchdowns.

= x:18 or x/18

Solution:

Let us use the fraction form so it will be easier for us to solve.

To find the number of touchdowns Timberwolves did, we need to cross multiply.

1/6 = x/18

6x = 18

6x/6 = 18/6

x = 3

Final Answer:

Timberwolves did 3 touchdowns if the cheerleaders set off 18 confetti cannons.
